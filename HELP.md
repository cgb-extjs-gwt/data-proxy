# 数据代理

## 初始化项目

npm init

## 扩展

### typescript
npm install --save-dev typescript

### lodash

npm i --save lodash

## 发布

::发布前需要把npm源切回npm
npm config set registry https://registry.npmjs.org

::登录  
npm login

::上传  
npm publish

::npm源切回淘宝
npm config set registry https://registry.npm.taobao.org
